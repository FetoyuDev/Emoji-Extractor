# Avvertenza

- **Questo strumento richiede curl installato, altrimenti, questo strumento non funzionerÃ **
- **Ma per fortuna tua, ti mostrerÃ² come installare curl in diverse distribuzioni**

### Per distribuzioni basate su Ubuntu/Debian
```bash
sudo apt update -y && sudo apt install curl git -y
```

### Per distribuzioni basate su Fedora/CentOS/Red Hat
- se il sistema usa "yum"
```bash
sudo yum install curl git -y
```

- Se il sistema usa "DNF"
```bash
sudo dnf install curl git -y
```

### Per distribuzioni basate su Arch Linux
```bash
sudo pacman install -Sy curl git --noconfirm
```

### Per distribuzioni basate su openSUSE
```bash
sudo zypper install curl git -y
```

## Configurazione (Per utenti Desktop o Laptop)

### Passo 1: Scarica e configura lo strumento
- 1. Apri il terminale.
- 2. Esegui il seguente comando per clonare il repository e configurare lo strumento:

```bash
git clone https://github.com/FetoyuDev/Emoji-Extractor.git ~ && cd Emoji-Extractor && sudo chmod +X init.sh && sh init.sh
```

### Passo 2: Abilita il comando `emojiextractor`

- 1. Per rendere disponibile il comando emojiextractor nel tuo terminale, esegui il seguente comando:
```bash
echo 'export PATH="$HOME/Emoji-Extractor:$PATH"' >> "$HOME/.bashrc" && ln -sf "$HOME/Emoji-Extractor/start.sh" "$HOME/Emoji-Extractor/emojiextractor" && chmod +x "$HOME/Emoji-Extractor/start.sh" && source "$HOME/.bashrc"
```
